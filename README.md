Healthcare Data Analysis Dashboard

Project Overview

The Healthcare Data Analysis Dashboard is an interactive Power BI dashboard designed to analyze healthcare business performance using key metrics such as revenue, profit, quantity, discounts, ratings, regional performance, and service categories. The dashboard enables users to monitor KPIs and identify trends for better operational and business decisions.

Objective

The primary objective of this project is to:

- Analyze overall healthcare revenue and profit
- Monitor daily profit trends
- Compare revenue across regions and categories
- Evaluate discounts and customer ratings
- Track service status distribution
- Identify high-performing regions and categories
- Support business decisions through interactive visualizations

Tools Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

Dashboard Features

-  Total Revenue KPI
-  Total Quantity KPI
-  Average Discount KPI
-  Day-wise Profit Analysis
-  Revenue by Region and Category
-  Region-wise Revenue Comparison
-  Category-wise Revenue Treemap
- Discount vs Rating Analysis
-  Record Status Distribution
-  Interactive Category Filters

Key Insights

- The dashboard generated **4.62M** in total revenue.
- Daily profit fluctuates, with several high-profit peaks throughout the month.
- Northern and Southern regions contribute significantly to revenue.
- Category-level analysis highlights the highest revenue-generating healthcare services.
- Customer ratings generally improve with moderate discount levels.
- Record status distribution provides visibility into completed, pending, and cancelled cases.
